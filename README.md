# Project Title

Its a Calculator to calculate how many dollars you have to pay as tip.

## Run Locally

Clone the project

```bash
  git clone https://github.com/thehacktour/tipcalculator
```

Go to the project directory

```bash
  cd mobile
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npx expo start
```
